How commissions are calculated

Commission = max(contract value × rate%, min commission)
Additional fees are added on top of the commission.
GST is applied to commission only (not additional fees).
Sell-side tax applies on sell orders only.
Break-even and P&L calculations include both buy and estimated sell commission.

Alert Proximity Warning
What is the Alert Proximity Warning %?

Controls how close the price must get to your alert target before showing a "NEAR HI" or "NEAR LO" warning.

Example: Set to 5% with HIGH alert at $100 → warning appears at $95 or above.

Set to 0% to only warn at the exact price.

ⓘ README — FORK & MAKE IT YOURS
A starting point, not a finished product.
This tracker is an open template. It's a single self-contained HTML file — no install, no backend, your data stays in the file. Setup is one click: open ⚙ Broker → pick your country and broker from the dropdown (iOCBC, UOBKH, Maybank, StanChart, Tiger, Moomoo, IBKR, and more — all with commission rates and FX spreads pre-filled) → Apply Preset. That's it. The fee fields populate automatically. Fine-tune individual fields later only if your statement shows different numbers. Fork it, change anything, ship your own version.
🛠 What people commonly customize
• Broker & rates — pick from preset list in ⚙ Broker (one click). Fine-tune individual fees only if your real statement disagrees (⚙ Broker has its own AI prompt to help you reconcile)
• Reporting currency — switch via the picker in the header / settings (default SGD, supports any)
• Visual theme — change CSS variables at the top: --accent, --gain, --loss
• Price provider — Finnhub / Alpha Vantage / manual are wired in; add Yahoo, IEX, your own API
• Columns & metrics — add Sharpe, Beta, dividend yield etc. to the CDEFS array + rendering pipeline
• Alerts — extend beyond H/L to RSI, MACD, news triggers

🛡 Profit Guard

Shows only positions where your current price is 15%+ above break-even — i.e. you're sitting on meaningful profit.

For each, it calculates:
• Trailing Stop — −5% from the session peak price. If price falls below this, consider locking in profits.
• TP1 — estimated next take-profit target at 30% more upside.

🚨 turns red when price is within 2% of the trailing stop.
